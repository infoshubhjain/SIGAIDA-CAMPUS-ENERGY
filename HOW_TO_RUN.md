# How to Run the App

## Mac/Linux

1. Open Terminal
2. Navigate to the project folder:
   ```bash
   cd "/Users/shubh/Desktop/sigaida rep/SIGAIDA-CAMPUS-ENERGY"
   ```
3. Run the startup script:
   ```bash
   ./start.sh
   ```
4. The app will automatically open in your browser at http://localhost:3000

**To stop the app:**
```bash
./stop.sh
```

---

## Windows

1. Open the project folder in File Explorer
2. Double-click `start.bat`

   **OR** open Command Prompt and run:
   ```bash
   cd "C:\path\to\SIGAIDA-CAMPUS-ENERGY"
   start.bat
   ```
3. The app will automatically open in your browser at http://localhost:3000

**To stop the app:**
- Double-click `stop.bat`
- Or close the console windows

---

## That's it!

The app will be available at:
- **Dashboard**: http://localhost:3000
- **API**: http://localhost:8000
- **API Docs**: http://localhost:8000/api/docs
